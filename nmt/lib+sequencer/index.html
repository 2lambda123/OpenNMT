nmt.Sequencer

Sequencer is the base class for our time series LSTM models. Acts similarly to an nn.Module. Main task is to manage self.network_clones, the unrolled LSTM used during training. Classes encoder/decoder/biencoder generalize these definitions.

Inherits from Model.
